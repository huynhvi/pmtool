# Scenario – Goal Setting Dashboard (Mass Roll-out)

## Context
PM Tool is currently in the Mass Roll-out phase.
A large number of goal sheets are being processed across multiple departments and approvers.

## Problem
The PM cannot quickly answer:
- How many goal sheets are completed?
- Which departments are behind?
- Which approvers are bottlenecks?
- Who has not started yet?

This leads to slow follow-up and risk of missing the Goal Setting timeline.

## Expected Solution
A simple single-page dashboard that allows:
- Upload Excel file
- Automatically calculate key metrics
- Display progress by:
  - Status
  - Department
  - Approver

## Daily Usage
PM will:
- Check overall progress
- Identify delayed departments
- Identify overloaded approvers
- Filter follow-up list

## Expected Outcome
- Understand rollout status within seconds
- Identify who needs follow-up immediately
- Eliminate manual Excel tracking

## Success Criteria
- Single page only
- Easy to read
- Action-oriented (supports follow-up)
